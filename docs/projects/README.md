# Projects Directory

This directory contains all the individual web apps, animations, and experiments.

## Structure

Each project should be contained in its own subdirectory with the following structure:

```
projects/
├── project-name/
│   ├── index.html
│   ├── styles.css (optional)
│   ├── script.js (optional)
│   └── README.md (optional)
```

## Adding a New Project

1. Create a new directory in `projects/` with a descriptive name (use kebab-case)
2. Add your project files (at minimum an `index.html`)
3. Update the main `../index.html` to link to your project
4. Commit and push your changes
